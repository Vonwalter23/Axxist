# STAGE_09_ANDROID_ACTIONS

## Objetivo

Implementar las primeras acciones reales utilizando Intents oficiales de Android.

---

## Implementar

- Abrir aplicaciones
- Abrir configuración
- Abrir cámara
- Abrir galería
- Abrir navegador
- Abrir mapas
- Compartir texto

---

## Restricciones

Utilizar únicamente APIs oficiales.

No usar Accessibility Service.

No realizar automatizaciones invasivas.
