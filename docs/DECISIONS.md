# DECISIONS

Este documento registra únicamente decisiones de arquitectura ya aprobadas.

---

## 2026-07-16

- Nombre del proyecto: Axxist.
- Desarrollo incremental por Stages.
- React Native + Kotlin.
- JDK 17 obligatorio.
- TypeScript obligatorio.
- Arquitectura modular.
- Runtime permanente.
- Android 10+.
- AI Router como única puerta de acceso a cualquier IA.
- Groq será el primer proveedor de IA.
- Soporte futuro para IA Local (Qwen, Gemma, Phi y SmolLM).
- Todos los cambios deberán documentarse.
- Cada Stage deberá generar una APK.
- Cada Stage deberá generar un Reporte.
- Cada Stage deberá actualizar el CHANGELOG.
- Cada Stage deberá crear commits descriptivos.
- Cada Stage deberá subir los cambios a GitHub.
- Cada Stage deberá generar una GitHub Release.
- Los Stages aprobados no se modifican. Las nuevas funcionalidades se incorporan mediante nuevos Stages.
